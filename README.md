## C++ Weather Application ##

This is a console weather application written in c++. It fetches weather information from the OpenWeatherApi using the REST Api Architecture 


### Requirements ###
* vcpkg

### Required modules ###
* cpprestsdk


### Run ###
    cd x64\Debug
    c++ mini project.exe
